# simple cli demo project


## how to running

* init venv

```code
python3 -m venv .
```

* install click pacakge

* activate venv

```code
source bin/activate
```

* install click pacakge

```code
pip install click
```

* install with pip pacakge for test

```code
pip install .
```

* test cli

```code
dalongcli --name dalong --count 3
```

result

```code
Hello, dalong!
Hello, dalong!
```
